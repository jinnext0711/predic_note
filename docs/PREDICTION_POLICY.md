# 予想方針書（Prediction Policy）

本書は Agent Team の予想方針を定める。全エージェントはこの方針に従って分析・記事作成を行う。

---

## 1. 基本スタンス：バランス型

レースの性質に応じて**攻め（高配当狙い）と守り（的中重視）を使い分ける**。
**最重要ルール: トリガミ（的中しても収支マイナス）を絶対に起こさない馬券設計にする。**

| レースタイプ | 戦略 | 狙い |
|-------------|------|------|
| 堅いレース（自信度A-B） | 的中重視 → 単勝・馬単・三連単（本命流し） | 確実に当てて信頼を積む |
| 混戦レース（自信度B-C） | バランス → ワイド・三連複（ボックス/フォーメーション） | 的中率と回収率の両立 |
| 波乱含みレース（自信度C-D） | 回収重視 → 三連単・馬単（穴馬からの流し） | 高配当で大きく回収 |
| 難解レース（自信度D） | 見送り or 最小投資 | 無理しない |

**長期目標**: 回収率100%超を安定的に達成する。的中率だけを追わない。

---

## 2. 予算設定

### 1レースあたり：最大 10,000円

| 自信度 | 投資額目安 | 方針 |
|--------|-----------|------|
| A（自信あり） | 8,000〜10,000円 | 厚めに張る。勝負レース |
| B（やや自信） | 5,000〜8,000円 | 標準投資 |
| C（五分五分） | 2,000〜5,000円 | 控えめ。ワイド中心 |
| D（難解） | 0〜1,000円 | 見送り推奨。買うなら複勝のみ |

### 馬券配分ルール（トリガミ防止設計）

**大原則: 的中した場合に必ずプラス収支になる馬券構成にする**

- **攻め（勝ち意識）**: 投資額の30〜50%
  - 馬単、三連単など高配当を狙う券種
  - 外れても守りでカバーできる範囲内
- **守り（回収保証）**: 投資額の50〜70%
  - ワイド、三連複、複勝など的中率の高い券種
  - **守りの馬券単体で投資総額を回収できる設計**にする

### トリガミ防止チェックリスト
1. **複勝の払戻し > 全馬券の投資額** を必ず確認
2. ワイドは最低オッズでもプラスになる金額設定
3. 三連複ボックスは点数×100円 < 想定最低払戻し
4. 馬単・三連単は「当たればボーナス」の位置づけ。外れても守りで回収
5. **トリガミリスクがある場合は買い目を減らす or 金額を調整**

## 3. 対象レースと分析の深さ

**全レースを予想対象とする。** ただし、レースの性質に応じて分析の深さを3段階に分ける。

### Tier 1：重賞レース（G1〜G3、リステッド）
- **フル分析**：6軸評価 + 4専門家レビュー + 全馬評価表 + 期待値ギャップ分析
- 記事の主役。読者の注目度が最も高い
- 予算は上限まで使う
- 記事内で最も紙面を割く

### Tier 2：注目レース（妙味のある平場）
- **詳細分析**：6軸評価 + 全馬評価表 + 推奨買い目
- 以下の条件に該当するレースを自動で「注目レース」に昇格：

| 注目レース基準 | 説明 |
|--------------|------|
| 期待値ギャップが大きい | 実力と人気の乖離が顕著な馬がいる |
| 波乱度「波乱含み」以上 | 荒れやすい構造で高配当のチャンス |
| 自信度A | データ的に読みやすく的中を取りに行ける |
| 条件替わりの妙味 | 距離変更、初ダート、コース替わりで激走期待の馬がいる |
| 血統・調教の特注馬 | 専門家が太鼓判を押す穴馬がいる |
| 特別戦 | ○○ステークス等の名前付きレース |

### Tier 3：通常レース（その他の平場）
- **基本分析**：印（◎○▲△）+ 推奨買い目 + 一言見解
- 全馬評価表は簡略版（総合指数のみ）
- コンパクトに掲載。読者が買い目だけ確認できる形式

## 4. 差別化：独自指数 + 専門家解説

### 独自指数システム「PredIndex」

各馬を **6軸×100点満点** で数値化した独自指数を全馬に付与する。

```
PredIndex = 能力(30%) + 調子(15%) + 騎手(15%) + 展開(15%) + 血統(15%) + 厩舎(10%)
```

| 指数帯 | グレード | 意味 |
|--------|---------|------|
| 80〜100 | A | この条件で最上位。軸候補 |
| 60〜79 | B | 好走圏内。相手候補 |
| 40〜59 | C | 平均的。展開次第 |
| 20〜39 | D | やや厳しい。よほどの展開利がないと |
| 0〜19 | E | 厳しい。消しの対象 |

**読者への見せ方**: 表で全馬の6軸スコアを公開し、なぜその数値なのかを解説する。
「なんとなく」ではなく「データに基づく根拠」を示すことで信頼を得る。

### 4専門家レビューの融合

初期分析を**4人の専門家がクロスチェック**して精度を高める。

| 専門家 | 補正する軸 | 記事での見せ方 |
|--------|-----------|--------------|
| 競馬場専門家 | 展開・枠順 | 「コース傾向メモ」として掲載 |
| 熟練予想家 | 印・波乱度 | 「ベテランの眼」として見解を補足 |
| 血統専門家 | 血統スコア | 「血統PICK」として特注馬を紹介 |
| 調教専門家 | 調子・厩舎 | 「調教CHECK」として仕上がり評価 |

**読者に伝えるメッセージ**:
> 「AI分析 × 専門家の目 × 独自指数。データと経験の融合で、根拠のある予想を提供します」

## 5. 印のルール

| 印 | 意味 | 付与基準 |
|----|------|---------|
| ◎ 本命 | 最も勝つ可能性が高い | PredIndex 1位 + 自信あり |
| ○ 対抗 | 本命を脅かす存在 | PredIndex 2位 |
| ▲ 単穴 | 一発があるかも | PredIndex 3位 or 期待値ギャップ大 |
| △ 連下 | 馬券圏内の可能性あり | PredIndex 4〜6位で指数差が小さい |
| × 消し | 人気だが危険 | 人気上位だがPredIndex低い（危険な人気馬） |
| 無印 | 馬券対象外 | 今回は厳しい |

**印のポリシー**:
- ◎は**必ず1頭**（複数つけない）
- ×（消し）は**人気馬にのみ付ける**（不人気馬を消す意味はない）
- 全馬に印をつける必要はない。メリハリをつける

---

## 6. 危険な人気馬の基準

以下に該当する馬は、人気でも「×消し」候補とする：

- PredIndex が人気順位より**3ランク以上低い**
- 前走好走の**反動リスク**（G1激走直後の格下げ戦など）
- **距離延長/短縮**で明らかに適性外
- **初コース・初距離**で血統的にも合わない
- 乗り替わりで**テン乗り騎手**（コース経験なし）
- **休み明け**で仕上がり不十分な厩舎パターン

## 7. 記事のトーン

### 目指す読者像
- 競馬歴1〜5年の中級者がメイン
- データに興味があるが、専門的すぎると読めない層
- 「理由があるから買う」を求めている人

### 文体ルール
- **断定しすぎない**: 「○○が勝ちます」→「○○に期待したい」
- **根拠を必ず添える**: 「穴で面白い」→「PredIndex 72だが8番人気。前走の上り33.0秒は出走馬中トップ。期待値1.8」
- **危険を明示**: 「人気のA馬は消し」→理由を3行以内で
- **禁句**: 「的中保証」「絶対」「必ず儲かる」
- **免責**: 記事末尾に「投資は自己責任」の免責文を必ず入れる
