# predic_keiba 開発環境

## このフォルダで開発する場合（すでにクローン済み）

1. **仮想環境を有効化**
   ```bash
   cd /Users/jinnobuta/test/predic_keiba
   source venv/bin/activate
   ```

2. **GitHub の最新を取り込む**
   ```bash
   git pull origin master
   # または main ブランチの場合: git pull origin main
   ```

3. **依存の追加後**
   ```bash
   pip install -r requirements.txt
   ```

## GitHub にまだリポジトリがない場合

1. https://github.com/new でリポジトリ名を `predic_keiba` にして作成
2. ローカルからプッシュ:
   ```bash
   cd /Users/jinnobuta/test/predic_keiba
   git push -u origin master
   ```

## 認証エラーが出る場合

- **HTTPS**: Personal Access Token (PAT) を使用
- **SSH**: `git remote set-url origin git@github.com:jinnext0711/predic_keiba.git` で SSH に切り替え
